## ğŸ“– Core Concepts

### Devices
Network devices (routers, switches, servers, PCs) that are discovered and monitored.

### Ports
Physical or virtual network interfaces on devices.

### Links
Connections between devices, representing physical or logical network paths.

### Events
System events and alerts generated by monitoring activities.

### Cable Health
Metrics and diagnostics about physical cable quality and performance.
